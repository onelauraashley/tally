# Rally — Product Brief

## Vision
Rally is a cozy, warm game night companion that makes it effortless to track scores, settle rule disputes, and celebrate who's actually winning the long game across your friend group.

## Target audience
- Primary: Adults 25–45 who regularly play board, card, and lawn games with friends or family
- Secondary: Casual gamers who want something more organized than pen and paper but less serious than a spreadsheet
- Geographic: English-speaking markets, US first

## App name
Rally — a tally mark is the icon motif. Simple, universal, readable at any size.

## Platform
- Web first (responsive, deployed on Vercel)
- Mobile PWA second (installable, works on phone home screen)
- No native app store distribution planned

## Design direction
Cozy, warm, board game night. Think a well-lit kitchen table, worn wooden game boxes, hot drinks in mismatched mugs. Modern and innovative but not clinical, not SaaS, not AI-looking, not neon gaming. iOS liquid glass aesthetic — frosted panels, subtle elevation, layered depth.

## Color story
Two colors: violet for structure, teal for celebration. Same palette in light and dark mode. Violet handles navigation, CTAs, selected states, and structural elements. Teal handles wins, streaks, score deltas, rank #1, and all celebration moments.

## Typography
Nunito 900 for all display moments — scores, titles, game names, buttons.
Open Sans 400/600 for all UI text — labels, metadata, inputs, timestamps.

## Core value propositions
1. Easiest way to track scores at game night — no pen, no paper, no arguments
2. Rule disputes settled instantly via AI-powered rules chat scoped to the specific game
3. Leaderboards that show who's actually the best across your friend group over time

## Monetization
Free with soft rate limit on AI rules chat (10 questions per user per day).
Phase 2: Rally Pro at $2.99/month for unlimited AI chat, shareable score cards, extended history, profile themes.
Core features (scoring, leaderboards, friends, preset games) stay free forever.

## Success metrics
- Users complete at least one full game session
- Users return for a second session within 7 days
- Leaderboard is checked between game nights, not just during
